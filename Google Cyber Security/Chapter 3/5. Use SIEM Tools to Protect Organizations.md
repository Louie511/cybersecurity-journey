## 🔷 This lecture focuses on how Security Information and Event Management (SIEM) dashboards are used by cybersecurity professionals to identify and analyze potential threats, risks, and vulnerabilities within an organization's systems.

## 🔺 Splunk SIEM Dashboards

### Security Posture Dashboard: 
- Designed for Security Operations Centers (SOCs), this dashboard displays recent security events and trends, helping analysts monitor and investigate potential threats in real time.

### Executive Summary Dashboard: 
- This dashboard provides a high-level overview of an organization's security health over time, assisting security teams in improving measures and offering insights to stakeholders.

## Chronicle SIEM Dashboards

### Enterprise Insights Dashboard: 
- This dashboard highlights recent alerts and identifies suspicious domain names (indicators of compromise), providing confidence scores and severity levels for potential threats.

### IOC Matches Dashboard: 
- This dashboard shows the top threats, risks, and vulnerabilities, allowing security professionals to observe trends in domain names, IP addresses, and device indicators of compromise to prioritize their efforts.
