## 🔷 This lecture explains how networks facilitate communication and connection within organizations, while also highlighting the security risks associated with data transfer.

## 🔺 Data Packets: The Basics of Network Communication

- Data is transferred across a network in basic units called data packets, which contain information about the sender, receiver, and message content.
- Similar to a physical letter, a data packet has a header (with IP and MAC addresses), a body (the message), and a footer (signaling the end of the packet).

## 🔺 Network Performance and Security

- Network performance can be measured by bandwidth (amount of data received per second) and speed (rate of packet reception).
- Irregularities in bandwidth or speed can indicate a network attack, and packet sniffing is a technique used to capture and inspect data packets for security purposes.
