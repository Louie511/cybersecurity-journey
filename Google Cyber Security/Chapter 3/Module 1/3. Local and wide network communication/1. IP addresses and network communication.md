## 🔷 This lecture covers how IP addresses and MAC addresses are used for communication within a network.

## 🔺 IP Addresses: Identification and Types

- An IP address is a unique string of characters that identifies a device's location on the internet, similar to a mailing address for a house.
- here are two main types: IPv4, which uses four sets of 1-3 digit numbers, and IPv6, a newer version with 32 characters developed to accommodate the growing number of internet-connected devices.

## 🔺 Public vs. Private IP Addresses

- A public IP address is assigned by your internet service provider and is shared by all devices on your network when communicating with the internet.
- Private IP addresses are used for communication between devices within the same local network and are not visible to the rest of the internet.

## 🔺 MAC Addresses: Device Identification

- A MAC address is a unique alphanumeric identifier assigned to each physical device on a network.
- Switches use MAC addresses to direct data packets to the correct device by maintaining a MAC address table, which acts like an address book.
