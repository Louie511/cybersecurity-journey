## 🔷 This lecture explains the TCP/IP model, which is the standard for network communication, by detailing the functions of TCP and IP separately.

## 🔺 Transmission Control Protocol (TCP)

- TCP is an internet communication protocol that enables two devices to establish a connection and stream data.
- It provides instructions for organizing data for network transmission and ensures data packets reach their intended destination.

## 🔺 Internet Protocol (IP)

- IP consists of standards for routing and addressing data packets as they move between devices on a network.
- An IP address, included in the Internet Protocol, serves as a unique address for each private network.

## 🔺 Ports and Data Packet Handling

- Ports are software-based locations within a network device's operating system that organize data sending and receiving.
- They divide network traffic into segments, allowing computers to prioritize and process data based on port numbers, similar to apartment numbers in a building.
