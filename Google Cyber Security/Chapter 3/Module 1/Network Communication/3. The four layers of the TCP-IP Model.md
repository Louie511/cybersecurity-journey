## 🔷 This lecture explains the TCP/IP model, a framework used by security professionals to visualize how data is organized and transmitted across a network.

## 🔺 TCP/IP Model Layers

### ⚫ The TCP/IP model consists of four layers: 
1. Network access
2. Internet
3. Transport
4. Application.

- Understanding these layers helps security professionals monitor and secure against risks.

## 🔺 Network Access and Internet Layers

- The network access layer handles the creation and transmission of data packets, involving hardware like cables and switches.
- The internet layer attaches IP addresses to data packets, indicating sender and receiver locations, and determines how networks connect.

## 🔺 Transport and Application Layers

- The transport layer uses protocols to control traffic flow, permitting or denying communication and ensuring smooth data transfer through error control.
- The application layer defines how data packets interact with receiving devices, supporting functions like file transfers and email services.
