## 🔷 This lecture provides an overview of the TCP/IP model, its layers, and common protocols, highlighting its importance for security professionals.

## 🔺 TCP/IP Model Fundamentals

- The TCP/IP model is a framework for visualizing how data is organized and transmitted across a network.
- It helps network engineers and security analysts understand network processes and identify disruptions or security threats.

## 🔺 Layers of the TCP/IP Model

### ⚫ Network Access Layer: This layer handles the creation and transmission of data packets, involving physical hardware like hubs, modems, and cables.
### ⚫Internet Layer: Responsible for ensuring data delivery to the correct destination, even on different networks, using IP addresses and protocols like IP and ICMP.
### ⚫Transport Layer: Delivers data between systems or networks, controlling traffic flow with protocols such as TCP (reliable, connection-oriented) and UDP (connectionless, performance-sensitive).
### ⚫Application Layer: Defines internet services and applications users can access, with protocols like HTTP, SMTP, SSH, FTP, and DNS.
