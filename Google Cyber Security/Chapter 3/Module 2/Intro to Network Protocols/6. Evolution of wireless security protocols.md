## 🔷 This lecture explores the evolution of wireless security protocols, highlighting their development from early, less secure versions to the more robust standards used today.

## 🔺 Early Wireless Security: WEP

- WEP (Wired Equivalent Privacy) was the first wireless security protocol, introduced in 1999, aiming to provide wired-level privacy for wireless networks.
- Despite its initial purpose, WEP is now considered high-risk due to vulnerabilities that allow malicious actors to break its encryption.

## 🔺 WPA: A Transitional Improvement

- WPA (Wi-Fi Protected Access) was developed in 2003 as an improvement over WEP, addressing its security flaws and serving as a transitional measure.
- WPA introduced TKIP (Temporal Key Integrity Protocol) and larger encryption keys, along with a message integrity check to detect tampering.

## 🔺 WPA2 and WPA3: Enhanced Security Standards

- WPA2, released in 2004, further enhanced security by using AES (Advanced Encryption Standard) and CCMP (Counter Mode Cipher Block Chaining Message Authentication Code Protocol).
- WPA3, introduced in 2018, addresses the KRACK attack vulnerability present in WPA2 and uses Simultaneous Authentication of Equals (SAE) for stronger password protection and increased encryption.
