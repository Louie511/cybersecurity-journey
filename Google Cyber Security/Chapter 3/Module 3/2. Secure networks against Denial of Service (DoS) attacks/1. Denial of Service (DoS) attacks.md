## 🔷 This lecture discusses Denial of Service (DoS) attacks, which aim to disrupt normal business operations by overwhelming a network or server with traffic.

## 🔺 Understanding Denial of Service (DoS) Attacks

- A DoS attack floods a network or server with traffic, making it unable to respond to legitimate users and potentially causing it to crash.
- A Distributed Denial of Service (DDoS) attack uses multiple devices from different locations to flood the target, increasing the likelihood of overwhelming the server.

## 🔺 Common Network-Level DoS Attacks

### ⚫ SYN Flood Attack: 
- This attack exploits the TCP connection handshake by sending numerous SYN requests to a server. If the number of requests exceeds the server's available ports, it becomes overwhelmed and unable to function.

### ⚫ ICMP Flood Attack: 
- Attackers repeatedly send ICMP packets to a server, forcing it to send ICMP responses. This consumes bandwidth for both incoming and outgoing traffic, leading to a server crash.

## 🔺 Ping of Death Attack

- This DoS attack involves sending an oversized ICMP packet (larger than 64 kilobytes) to a vulnerable network server. The large packet overloads the system, causing it to crash.
