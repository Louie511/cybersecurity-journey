## 🔷 This lecture explains network hardening, which focuses on securing networks through various practices.

## 🔺 Regularly Performed Network Hardening Tasks

### ⚫ Firewall Rules Maintenance and Log Analysis: 
- Regularly updating firewall rules and analyzing network logs are crucial for identifying and mitigating security events.

### ⚫ Patch Updates and Server Backups: 
- Consistent application of patch updates and regular server backups help maintain network security and data integrity.

## 🔺 One-Time Network Hardening Tasks

### ⚫ Port Filtering and Network Access 
- Privileges: Implementing port filtering on firewalls to restrict unwanted communication and carefully managing network access privileges are essential for initial setup.

### ⚫ Network Segmentation and Encryption: 
- Segmenting networks to isolate different departments or security zones and encrypting all network communication using the latest standards enhance overall security.
