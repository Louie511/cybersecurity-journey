## 🔷 This lecture explains how cloud networks are secured, contrasting them with traditional networks and highlighting the shared responsibilities in cloud security.

## 🔺 Securing Cloud Networks

- Cloud networks, like on-premises networks, require security measures due to their role in storing resources and data in remote data centers.
- Cloud service providers and organizations share responsibility for security, with organizations needing to implement their own measures.

## 🔺 Cloud Network Hardening

- A key distinction in cloud network hardening is the use of a server baseline image to detect unverified changes, which could indicate an intrusion.
- Data and applications within a cloud network should be separated based on their service category, similar to OS hardening, to enhance security.
