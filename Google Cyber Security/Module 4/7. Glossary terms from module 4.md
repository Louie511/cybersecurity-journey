## This lecture provides definitions for key terms related to cybersecurity tools and concepts.

## Cybersecurity Software and Systems

### Antivirus software: 
- Programs designed to prevent, detect, and remove malicious software and viruses.

### Intrusion detection system (IDS): 
- An application that monitors system activity and alerts on potential intrusions.

## Data Management and Analysis

### Database: 
- An organized collection of information.

### SQL (Structured Query Language): 
- A language used to create, interact with, and request information from databases.

## System Monitoring and Forensics

### Log: 
- A record of events occurring within an organization's systems.

### Order of volatility: 
- The sequence in which digital data should be preserved, from most to least volatile.
